# 🏠 HouseHunt – AI-Enabled Smart House Rental Platform

HouseHunt is a Full Stack MERN (MongoDB, Express.js, React.js, Node.js) web application that simplifies the process of finding rental properties. The platform allows users to search for houses, view detailed property information, and receive AI-powered assistance for discovering suitable homes based on their preferences.

## 📌 Project Overview

Finding a suitable rental property can be time-consuming due to scattered listings and limited search capabilities. HouseHunt provides a centralized platform where users can browse verified properties, apply filters, and interact with an AI assistant for personalized property recommendations.

## ✨ Features

- 👤 User Registration and Login
- 🏠 Browse Rental Properties
- 🔍 Search and Filter Houses
- ❤️ Save Favorite Properties
- 🤖 AI-Powered Property Recommendation Assistant
- 📱 Responsive User Interface
- 🔒 Secure Authentication
- ⚡ Fast and Modern MERN Stack Architecture

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Hema-0508/HouseHunt.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
